# Документация по установке DonkeyCar

## Созданные файлы

Для будущей установки созданы следующие файлы:

1. **`setup_donkeycar_full.sh`** - Полностью автоматический скрипт установки
   - Выполняет все шаги автоматически
   - Проверяет зависимости
   - Создает окружение
   - Устанавливает DonkeyCar

2. **`INSTALLATION.md`** - Подробная инструкция по установке
   - Описание всех шагов
   - Решение проблем
   - Ручная установка

3. **`QUICK_START.md`** - Быстрый старт
   - Краткие инструкции
   - Что передать чат-боту для установки

4. **`REQUIREMENTS.txt`** - Системные зависимости
   - Список всех необходимых библиотек
   - Команды для установки

## Использование для будущей установки

### Вариант 1: Автоматическая установка (рекомендуется)

```bash
cd ~/projects/donkeycar
./setup_donkeycar_full.sh
```

### Вариант 2: Передача чат-боту

Скажите чат-боту:
```
Установи DonkeyCar используя скрипт setup_donkeycar_full.sh 
в директории ~/projects/donkeycar
Или следуй инструкциям из INSTALLATION.md

Важно: Python 3.11.9, обязательно установить libffi-dev ДО установки Python!
```

## Ключевые требования (для справки)

- **Python**: 3.11.9 (строго, через pyenv)
- **Критическая библиотека**: libffi-dev (для ctypes, нужна ДО установки Python)
- **Виртуальное окружение**: env с Python 3.11.9
- **TensorFlow**: 2.15.1 (tensorflow-aarch64, работает только с Python 3.11)

## Порядок установки (ВАЖНО!)

1. ✅ Установить libffi-dev и другие системные зависимости
2. ✅ Установить pyenv
3. ✅ Установить Python 3.11.9 через pyenv
4. ✅ Создать виртуальное окружение с Python 3.11.9
5. ✅ Установить DonkeyCar: `pip install -e .[pi]`
