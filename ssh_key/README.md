# SSH ключ для GitHub

Этот каталог содержит публичный SSH ключ для работы с GitHub.

## Файлы

- `id_ed25519.pub` - Публичный SSH ключ (можно безопасно передавать)

## Как использовать

### Добавление ключа в GitHub

1. Скопируйте содержимое файла `id_ed25519.pub`:
   ```bash
   cat id_ed25519.pub
   ```

2. Перейдите на https://github.com/settings/keys

3. Нажмите "New SSH key"

4. Вставьте содержимое ключа и сохраните

### Проверка подключения

После добавления ключа в GitHub:
```bash
ssh -T git@github.com
```

## Важно

- ⚠️ **НИКОГДА не копируйте приватный ключ** (`~/.ssh/id_ed25519`) сюда или куда-либо еще!
- Публичный ключ (`id_ed25519.pub`) можно безопасно хранить и передавать
- Приватный ключ должен оставаться только в `~/.ssh/id_ed25519` с правами доступа 600

## Создание ключа

Ключ был создан командой:
```bash
ssh-keygen -t ed25519 -C "pi4_user@github" -f ~/.ssh/id_ed25519
```
